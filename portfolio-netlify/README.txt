FILES
- index.html     Main page
- styles.css     Styling
- script.js      Interactivity
- _redirects     Netlify SPA fallback
- resume.pdf     (ADD THIS) Your CV — referenced by the Download CV button

DEPLOY TO NETLIFY (drag & drop)
1. Drop your resume PDF into this folder and rename it to: resume.pdf
2. Go to https://app.netlify.com/drop
3. Drag this entire folder onto the page
4. Done — your site is live with a free *.netlify.app URL

DEPLOY TO GITHUB PAGES
1. Create a new repo, upload these files (with resume.pdf)
2. Repo Settings → Pages → Source: deploy from main branch / root
3. Visit https://<username>.github.io/<repo>

NOTES
- Contact form opens the visitor's email app (mailto). For real form
  submissions on Netlify, add `data-netlify="true"` to the <form> tag.
- Update the LinkedIn URL in index.html (search "https://linkedin.com").
- All meta tags, accessibility labels, and SEO basics are included.
